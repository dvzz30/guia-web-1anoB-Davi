O que é HTML, CSS e JS?

HTML

HTML é a linguagem de marcação que estrutura o conteúdo da web, definindo elementos como títulos, parágrafos, imagens e links.

<h1>Título</h1>
<p>Parágrafo</p>

Use para estruturar e organizar o conteúdo da página web.

CSS

CSS é a linguagem de estilização que controla a aparência e o layout dos elementos HTML.

.card {
    color: #111827;
    padding: 1rem;
}

Use para estilizar e criar layouts responsivos.

JavaScript

JavaScript é a linguagem de programação que adiciona interatividade e comportamento dinâmico às páginas web.

document.querySelector('.btn')
    .addEventListener('click', 
    () => alert('Clicou!'));

Use para adicionar interatividade e manipular dados.
